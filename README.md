# CondAdmin - Gestione cómoda y adecuadamente su conjunto habitacional

## Visión
Aplicativo Web para la gestión administrativa de un conjunto residencial
- Gestión de directiva
- Gestión de pagos
- Gestión de propietarios
- Gestón de gastos

## Tecnologias
- MySql
- H2
- SpringBoot
- thymeleaf
